## 제2장 핵심 기술의 독창성·차별성 논증

### 2.1 R 점수 산출 공식 — 법률적 판단의 수학적 모델링

#### 공식 정의

$$R = \sum_{i=1}^{n} \left( w_i \times \log_{10}(1 + f_i \times \overline{a}_i) \times T_i \times \Theta_i \times S_i \right)$$

| 기호 | 의미 | 범위 | 기술적 근거 |
|------|------|------|-------------|
| $w_i$ | 법적 민감도 가중치 | 0.1 ~ 5.0 | 채무자회생법 제564조 1~10호에 대응 |
| $f_i$ | 해당 지표 거래 건수 | 양의 정수 | 통합 데이터에서 지표별 집계 |
| $\overline{a}_i$ | 해당 지표 평균 거래 금액 | 양의 실수 | 출금 기준 평균 |
| $T_i$ | 시계열 근접성 계수 | 1.00 ~ 1.50 | 파산 직전 의심 패턴 탐지 |
| $\Theta_i$ | 동적 임계 계수 | 0.00 ~ 2.60 | 금액·건수 복합 가중 |
| $S_i$ | 입출금 대칭성 계수 | 0.30 ~ 1.00 | 회수 비율 반영 |

#### 왜 "단순 통계"가 아닌가 — 기보 평가관 설득 논리

**핵심 주장**: 본 R 공식은 단순한 통계적 합산이 아니라, **채무자회생법 제564조의 면책 불허가 사유 10개 항목을 수학적으로 모델링한 법률-수학 융합 알고리즘**이다.

1. **법적 가중치의 비선형 설계**: $w_i$는 법조항의 심각도에 따라 0.1(경미) ~ 5.0(사행성)으로 50배 차이를 두며, 이는 판례 분석에 기초한 법적 판단 기준을 수치화한 것이다.

2. **로그 스케일 왜곡 방지**: $\log_{10}(1 + f \times \overline{a})$는 단발성 고액 거래에 의한 위험도 왜곡을 방지하면서, 반복·소액 거래의 누적 위험을 적절히 반영하는 비선형 스케일링이다. 이는 단순 합산(Σ금액)이나 단순 평균(금액/건수)으로는 구현 불가능한 정밀 보정 기법이다.

3. **3차원 복합 보정 (T × Θ × S)**: 시간(T), 금액(Θ), 회수율(S)의 3차원 독립 변수가 곱연산으로 결합되어, **어느 하나의 축에서도 경미하면 전체 R이 감소**하는 자기 보정(Self-correction) 구조를 가진다. 이는 단순 가산 모델에서는 불가능한 다차원 위험 평가이다.

4. **Step-up Scoring의 법적 정합성**: 1호(0.1) → 10호(5.0)의 순차 덮어쓰기는 법원이 면책 불허가 사유를 판단하는 실무적 순서(경미한 사유에서 중대한 사유로)를 알고리즘으로 구현한 것이다. 상위 호수가 해당되면 하위 점수를 갱신(Overwriting)하여 이중 계산을 방지한다.

#### 자본 잠식 방어 논리 (기보 재무 평가 대응)

> **기보(KIBO)는 기술평가 시 재무제표도 병행 검토한다. 자본 잠식이 있는 초기 기업의 경우, 기술의 독보성과 시장성으로 재무적 결함을 상쇄해야 한다.**

| 재무 약점 | 기술 독보성 방어 | 구체적 논거 |
|---|---|---|
| 자본 잠식 | R 공식은 국내 유일의 **법률-수학 융합 모델** | 채무자회생법 10개 항목을 5개 수학 변수(w, f, ā, T·Θ·S)로 정량화한 사례는 선행기술 7건 조사에서 미발견 |
| 매출 부족 | **독점 가능한 기술 기반 매출 창출** 잠재력 | 특허 등록 시 법무사·변호사 사무소 대상 B2B SaaS 독점 가능, 개인파산 사건당 과금 모델 |

### 2.2 시계열 근접성 계수(T) — 선행기술 대비 차별점

#### T 계수의 정의 및 산출식

$$T = 1 + \alpha \times C + \beta \times P$$

| 변수 | 의미 | 산출식 | 기본값 |
|------|------|--------|--------|
| $C$ | 집중도 (Concentration) | $C = 1 - \frac{\text{활동기간}}{\text{전체기간}}$ | — |
| $P$ | 최근성 (Proximity) | $P = \frac{\text{최근 25\% 구간 건수}}{\text{전체 건수}}$ | — |
| $\alpha$ | 집중도 가중 | — | 0.3 |
| $\beta$ | 최근성 가중 | — | 0.2 |

#### 선행기술(문헌5, KR102486186B1)과의 차별점

| 비교 항목 | 문헌5 (AI 재무관리) | MyRisk T 계수 |
|---|---|---|
| **목적** | 기업 자금 흐름의 **미래 예측** | 파산 **직전** 의심 거래 패턴의 **사후 탐지** |
| **알고리즘** | ARIMA, LSTM, RNN | 선형 결합: $T = 1 + \alpha C + \beta P$ |
| **입력** | 기업 재무 시계열 데이터 | 개인 금융 거래 일자 배열 |
| **출력** | 미래 자금 흐름 예측값 | 1.00~1.50 범위의 승수 계수 |
| **기술적 사상** | **예측(Prediction)** — 미래 값 추정 | **진단(Diagnosis)** — 과거 패턴 이상 탐지 |
| **실무 의미** | "다음 달 자금이 부족할 것이다" | "파산 3개월 전 도박 지출이 급증했다" |

**기보 평가관 핵심 설득 포인트**: 문헌5의 시계열은 기업 재무의 미래를 예측하는 기술이고, MyRisk의 T는 **채무자의 과거 거래에서 법적으로 의심스러운 시간적 패턴(직전 집중, 단기 폭증)을 진단**하는 기술이다. 목적·알고리즘·출력 모두 근본적으로 상이하므로, T 계수는 **독자적 기술 창작물**이다.

#### 실무 시나리오 (기보 평가 보고서 활용)

> 채무자 A가 파산 신청 90일 전, 카지노 관련 출금 15건(총 800만원)을 집중적으로 실행했다고 가정한다.
>
> - 전체 분석 기간: 365일, 활동기간: 30일 → C = 1 − 30/365 ≈ 0.918
> - 최근 25%(91일) 내 건수 15건 / 전체 15건 → P = 1.0
> - T = 1 + 0.3 × 0.918 + 0.2 × 1.0 = **1.475**
>
> T가 1.0이면 R 점수 변동 없음. T = 1.475면 해당 지표의 R이 **47.5% 증가**하여, 파산 직전 집중 거래의 법적 의심도를 정량적으로 반영한다.

### 2.3 동적 임계 계수(Θ) — 금액·빈도 복합 가중

$$\Theta = \text{amount\_factor} \times \text{count\_factor}$$

| 구간 | amount_factor | 의미 |
|------|---------------|------|
| $a_{total} < threshold$ | $a_{total} / threshold$ | 금액 미달 → 선형 감점 |
| $threshold \le a_{total} \le 3 \times threshold$ | 1.0 | 기본 구간 |
| $a_{total} > 3 \times threshold$ | $1 + \gamma \times \log_2(a_{total}/threshold)$, 상한 2.0 | 누진 가중 |

| 구간 | count_factor | 의미 |
|------|--------------|------|
| $f < count\_threshold$ | 1.0 | 기본 |
| $f \ge count\_threshold$ | $1 + \delta \times (f/count\_threshold - 1)$, 상한 1.3 | 빈도 가중 |

**선행기술(문헌6, KR101914620B1) 대비 차별점**: 문헌6의 신뢰도 임계치는 ML 분류 결과의 **확실성을 평가**하는 메커니즘이나, Θ는 **법적 위험도의 크기를 금액+빈도 2축으로 보정**하는 승수이다. 목적(분류 확실성 vs 위험도 크기 보정)과 산출 방식이 근본적으로 상이하다.

### 2.4 입출금 대칭성 계수(S) — 투자 회수 반영

$$S = 1 - \frac{\text{입금합계}}{\text{출금합계}}, \quad \text{하한} = 0.3$$

- **S = 1.0**: 입금 = 0 (전액 유출) → R 변동 없음 (최대 위험)
- **S = 0.3**: 입금 ≈ 출금 (대부분 회수) → R **70% 감소** (위험 경감)
- 적용 대상: 투기성(5호), 가상자산(7호), 자산은닉(8호)만 해당

**기보 평가관 설득 포인트**: S 계수는 "가상자산에 1,000만원을 보냈지만 900만원을 회수했다면 순유출은 100만원에 불과하다"는 법적 실질 판단을 수학적으로 모델링한 것이다. 이 논리는 법원 실무에서 관재인이 실제로 수행하는 판단이며, 이를 자동화한 사례는 선행기술에서 미발견이다.

### 2.5 기술 결합의 진보성 — 종합 논증

| 핵심 기술 요소 | 선행기술 7건 대비 | 기보 평가 의미 |
|---|---|---|
| 3단계 계층적 분류 (전처리→계정과목→후처리) | 7건 중 **없음** | 비정형 금융 텍스트 분류의 새로운 패러다임 |
| 최장 일치 + 5단계 충돌 해결 캐스케이드 | 7건 중 **없음** (문7은 1:1 정적 매칭) | 동적 다기준 해결의 독자 알고리즘 |
| Step-up Scoring (1호→10호 순차 덮어쓰기) | 7건 중 **없음** | 법적 판단 순서의 알고리즘 구현 |
| R 점수 (T·Θ·S 복합 보정) | 7건 중 **없음** | 3차원 위험 평가의 자기 보정 모델 |
| VASP 3단계 식별 (카테고리→접두번호→사업자명) | 7건 중 **없음** | 가상자산 거래 탐지 전용 알고리즘 |
| 채무자회생법 제564조 자동 매핑 | 7건 중 **없음** | 법률-기술 융합의 유일한 사례 |
| 소명서 자동 생성 (템플릿+법조항) | 7건 중 **없음** | 법률 서비스 자동화 |

> **결론**: 상기 7개 핵심 기술 요소의 결합은 선행기술 7건 조사에서 어디에서도 개시(開示)되지 않았다. 개별 기술(금융 분류, 시계열 분석, 법률 문서 생성)은 각각 존재하나, **"채무자회생법 기반 + 정량적 위험도(R) + 다차원 보정(T·Θ·S) + 소명서 자동 생성"의 유기적 결합은 MyRisk가 유일**하다. 이는 기보 기술평가에서 **기술 독창성 최고 등급**에 해당하는 논거이다.
