### 진보성 판단

> 선행기술 7건은 법률 문서 분석/작성, 회계 분개, 가계부, AI 재무관리, ML 분개, 계정과목 분류에 특화. 문5~7은 "금융 거래 분류" 영역에서 겹치나 모두 **회계·세무** 또는 **기업 재무 예측** 목적.
>
> MyRisk 핵심 결합 — **이기종 통합 + 3단계 분류(최장 일치·5단계) + 채무자회생법 Step-up(1~10호) + R 점수(T·Θ·S) + 소명서 자동 생성** — 은 어떤 선행기술에서도 개시되지 않음.
>
> **진보성(비자명성) 확보에 유리**. 명세서에서 문5의 시계열 vs T 계수, 문6의 신뢰도 vs Θ 계수 차별점 명확 기술 권장.
