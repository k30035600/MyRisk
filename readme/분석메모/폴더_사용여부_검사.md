## 루트 직하위 폴더

| 폴더 | 사용 여부 | 비고 |
|------|-----------|------|
| **.cursor** | 사용 | Cursor 규칙·설정. 삭제 금지. |
| **.github** | 사용 | 이슈 템플릿, 워크플로(CI). |
| **.source** | 사용 | path_config.SOURCE_DIR, 원본 xls/xlsx. |
| **.venv** | 사용 | Python 가상환경. .gitignore. |
| **.vercel** | 설정 | 배포 설정. 앱 코드 미참조. |
| **.vscode** | 설정 | 에디터 설정. 앱 코드 미참조. |
| **data** | 사용 | path_config.DATA_DIR, bank/card/cash JSON, category_table. |
| **export** | 용도만 정의 | 분석·내보내기 결과용. 앱 코드에서 직접 참조 없음. (규칙: export/) |
| **lib** | 사용 | path_config, category_table_io 등. 서브앱에서 import. |
| **MyBank** | 사용 | app.py 서브앱 등록, 은행거래. |
| **MyCard** | 사용 | app.py 서브앱 등록, 신용카드. |
| **MyCash** | 사용 | app.py 서브앱 등록, 금융정보. |
| **readme** | 사용 | 문서·유틸. **삭제 금지** (규칙). |
| **temp** | 사용 | path_config.TEMP_DIR, 임시 파일. |
| **templates** | 사용 | app.py render_template (index, help, reset, 404). |

## .source 이하

| 폴더 | 사용 여부 | 비고 |
|------|-----------|------|
| **.source/Bank** | 사용 | SOURCE_BANK_DIR. 은행 원본 xls/xlsx. app.py 필수 디렉터리 생성. |
| **.source/Card** | 사용 | SOURCE_CARD_DIR. 카드 원본 xls/xlsx. app.py 필수 디렉터리 생성. |
| **.source/Cash** | 미사용 | 금융정보는 bank_after+card_after만 사용. app.py에서 **생성하지 않음**. (있으면 과거/수동 생성.) |
