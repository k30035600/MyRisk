# 위험도 지표 테이블 (1~10호)

`lib/category_table_io.py`의 `RISK_CLASS_TO_VALUE` 기준.

| 호수 | 위험도지표(위험도분류) | 위험도 |
|------|------------------------|--------|
| 1호 | 분류제외지표 | 0.1 |
| 2호 | 심야폐업지표 | 0.5 |
| 3호 | 자료소명지표 | 1.0 |
| 4호 | 비정형지표 | 1.5 |
| 5호 | 투기성지표 | 2.0 |
| 6호 | 사기파산지표 | 2.5 |
| 7호 | 가상자산지표 | 3.0 |
| 8호 | 자산은닉지표 | 3.5 |
| 9호 | 과소비지표 | 4.0 |
| 10호 | 사행성지표 | 5.0 |

- **위험도지표**: cash_after의 **위험도분류** 컬럼에 저장되는 값.
- **위험도**: 동일 행의 **위험도** 컬럼에 저장되는 수치(0.1 ~ 5.0).
- category_table.json에서 **분류** 또는 **카테고리**가 위 표의 위험도지표명인 행은 업종분류·위험도 매칭에 사용됨.
