# 업종분류·위험도 — category_table만 사용

## 1. 조사 개요

업종분류·위험도는 **category_table** 기반으로만 동작합니다. API는 `/api/risk-class-table`, 함수는 `get_risk_class_table_data`, `get_risk_class_map_for_apply`, `export_risk_class_table_to_xlsx`, `_apply_업종분류_from_category_table` 등입니다.

---

## 2. 반영된 수정 사항

### 2.1 Python 코드

| 위치 | 반영 내용 |
|------|-----------|
| **lib/category_table_io.py** | `get_risk_class_table_data`, `get_risk_class_map_for_apply`, `export_risk_class_table_to_xlsx`, `_default_risk_class_rows` 사용 (업종분류·위험도 전용) |
| **MyCash/cash_app.py** | `_apply_업종분류_from_category_table`, `get_risk_class_table()` → `/api/risk-class-table`, 위 함수들 직접 import |
| **readme/backup_업종분류_to_xlsx.py** | `export_risk_class_table_to_xlsx()` 사용 |

### 2.2 API·템플릿

- **`/api/risk-class-table`**: `get_risk_class_table_data()`로 응답. 클라이언트는 이 경로만 사용.
- 템플릿·문서: "category_table 기반 위험도 매칭", "업종분류 조회" 등으로 통일.

### 2.3 문서(readme)

- **위험도_호별_분류방법.md**, **업종분류_참조코드.md** 등: risk_class·category_table 기준으로 정리 완료.

## 3. 참고

- **data/업종분류_table.xlsx**: 백업·엑셀 편집용으로 **내보내기만** 유지. 생성 소스는 category_table 기반 (`export_risk_class_table_to_xlsx`).
