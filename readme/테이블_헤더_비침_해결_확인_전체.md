# Bank / Card / Cash 테이블별 "헤더 뒤로 행 비침" 해결 여부

## 해결 방식 구분

- **✅ 헤더/본문 분리 (split)**  
  스크롤 영역을 본문만 두고, 헤더는 별도 블록으로 고정. **비침 현상 해결됨.**
- **⚠️ 스티키만 사용 (sticky)**  
  `position: sticky` + z-index/box-shadow. 브라우저/환경에 따라 **비칠 수 있음.**
